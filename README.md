# 🍕 Online Food Ordering System

A dynamic web application built using **HTML, CSS, Bootstrap, PHP, and MySQL**.

## ✨ Features
- Menu display with categories and search filter
- Pagination for browsing large menus
- Add to cart and checkout system
- Order tracking functionality
- Role‑based Admin Panel:
  - **Normal Admin** → Add Items, manage orders/messages
  - **Superadmin** → Dashboard, manage admins, orders/messages

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL (phpMyAdmin)  
- **Server:** XAMPP  

## ⚙️ Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Shahzaib-Khan-106/Online_Food_Ordering_System.git
2. Import food_ordering.sql into phpMyAdmin.
3. Place the project in your XAMPP htdocs folder.
4. Run the project on localhost:
http://localhost/food-ordering/
