# 🍕 Online Food Ordering System

A dynamic web application built using **HTML, CSS, Bootstrap, PHP, and MySQL**.

## Features
- Menu display with categories
- Add to cart and checkout system
- Admin panel to add items with images
- Order tracking functionality

## Tech Stack
Frontend: HTML, CSS, Bootstrap  
Backend: PHP  
Database: MySQL  

## Setup
1. Clone the repository.
2. Import `food_ordering.sql` into phpMyAdmin.
3. Run the project on XAMPP: `http://localhost/food-ordering/`
